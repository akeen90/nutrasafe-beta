## ğŸ¨ Design Philosophy

### Emotion-First Language

We follow NutraSafe's premium onboarding aesthetic:

**Not:** "E102 is a synthetic color that may cause reactions"
**But:** "Can trigger hives and asthma in sensitive individuals"

**Not:** "Derived from insects"
**But:** "Made from 70,000 crushed female cochineal insects"

**Not:** "Regulatory concerns exist"
**But:** "WHO classified as 'possibly carcinogenic' in 2023"

### Glanceable Understanding

Users should understand the key facts in **3 seconds** by scanning colored bullets:

1. **Red bullets** = Stop and read carefully (danger/bans)
2. **Orange bullets** = Pay attention (warnings/concerns)
3. **Yellow bullets** = Be aware (cautions for some)
4. **Green bullets** = Context and facts (generally safe)

### Progressive Disclosure

1. **Collapsed:** Just the additive name + risk dot
2. **Expanded:** "What I need to know" (always visible)
3. **Deep dive:** Tap "Scientific Background" for full context
